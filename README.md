# 🧸 Claymorphism Portfolio - Gautham Krishna B

A soft, playful, and approachable portfolio design featuring extruded 3D shapes, bouncy animations, and a warm color palette.

## 🎨 Design Philosophy

Claymorphism combines soft 3D shapes with playful animations for a friendly, approachable interface. This portfolio uses:
- **Soft, extruded shapes** with rounded corners
- **Dual shadow system** (dark + light) for depth
- **Bouncy animations** with elastic easing
- **Warm, pastel color palette**
- **Card tilt effects** on mouse movement
